# Big Data Advanced
This project aim to implement the concepts of DMS NoSQL

## Comment exécuter notre projet
  * Création de la base de donnée avec le chargement des ddifférentes collections.
    Tout d'abord commencez par télécharger mongodb sur votre machine avec les utilitaires adéquats.
    Créez la base de donnée avec mongosh sous le nom de bankAgency.

    Chargez le fichier validations_database_crew.js présent dans scripts avec la commande load.
    Importez les collections depuis le dossier data_json avec la commande mongoimport.
